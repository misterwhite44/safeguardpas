Safe Guard Pass

Votre nouveau keepass indispensable pour sécuriser tous vos mots de passe. KeePass est un gestionnaire de mot de passe entièrement gratuit qui dispose de nombreuses fonctionnalités. Il permet aux utilisateurs d’enregistrer sur un même support, c’est-à-dire sur un seul un fichier entièrement protégé, leurs différents mots de passe utilisés sur Internet. Le code source de KeePass est ouvert à tous, ce qui fait que les développeurs peuvent régulièrement le mettre à jour. Il est téléchargeable à partir du site internet officiel et est disponible sur Windows, MacOs et Linux (pour ces deux derniers, l’installation se réalise via la plateforme Mono).
